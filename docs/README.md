# matrix-analysis
矩阵分析与 AI

在线阅读：https://liu-yang-maker.github.io/matrix-analysis/

[>>前往教程首页](index.md)

#### 项目初衷

本课程讲授和讨论矩阵分析的主要理论。

通过本课程的学习，希望能够掌握矩阵分析理论和主要思想，提高数理基础和思维水平。

学习的先修要求：了解一些基本的高数和线性代数知识。

#### 内容设置

1. 矩阵概述
   1. 向量、矩阵
   2. 矩阵的性能指标
   3. 矩阵的逆
   4. 广义逆和Moore-Penrose逆
   5. Kronecker积
   6. Hadamard积
2. 特殊矩阵
   1. 基本矩阵
   2. 置换矩阵
   3. 正交矩阵
   4. 相似矩阵
   5. Vandermonde矩阵
   6. Fourier矩阵
   7. Hadamard矩阵
3. [奇异值分析](chapter3.md)
   1. 数值稳定性和条件数
   2. 奇异值分解
   3. 奇异值分解的应用
   4. 广义奇异值分解
4. [最小二乘](chapter4.md)
   1. 最小二乘法
   2. 总体最小二乘
   3. 约束总体最小二乘
   4. 结构总体最小二乘
5. [子空间分析](chapter5.md)
   1. 子空间的一般理论
   2. 列空间、行空间、零空间
   3. 子空间方法
